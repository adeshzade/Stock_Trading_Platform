# 📈 Stock Trading Platform

A simple **Java console-based Stock Trading Platform** that simulates a basic stock market. Users can view available stocks, buy and sell shares, manage their investment portfolio, and track transaction history. This project demonstrates the use of **Object-Oriented Programming (OOP)** concepts in Java.

---

## 🚀 Features

- 📊 Display available stocks and current prices
- 💰 Buy stocks with balance validation
- 💸 Sell owned stocks
- 📁 Manage user portfolio
- 📝 Track transaction history
- 📈 Calculate total portfolio value
- 🖥️ Menu-driven console application
- 🔒 Uses Object-Oriented Programming (OOP)

---

## 🛠️ Technologies Used

## 📖 OOP Concepts Used

- **Encapsulation** – Private data members with getters and methods.
- **Abstraction** – Separate classes for stocks, users, and transactions.
- **Objects & Classes** – Real-world entities modeled as Java classes.
- **Collections** – HashMap for portfolio management and ArrayList for transaction history.

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/StockTradingPlatform.git
```

2. Open the project in IntelliJ IDEA, Eclipse, NetBeans, or VS Code.

3. Compile the Java files:

```bash
javac *.java
```

4. Run the program:

```bash
java StockTradingPlatform
```
